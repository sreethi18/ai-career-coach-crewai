# 🚀 AI Career Coach using CrewAI

An AI-powered multi-agent career assistant built with CrewAI, OpenAI, and Streamlit.

The platform analyses resumes, matches job opportunities, prepares interview questions, and generates a personalised 6-month career roadmap.

---

# ✨ Features

✅ Resume Analysis  
✅ Skill Extraction  
✅ Job Matching  
✅ Interview Preparation  
✅ Career Roadmap Generation  
✅ Streamlit Web Interface  
✅ Downloadable Career Report  

---

# 🧠 AI Agents

This project uses multiple CrewAI agents working together:

| Agent | Responsibility |
|---|---|
| Resume Analyst | Extracts skills and improves resumes |
| Job Matcher | Finds matching roles and skill gaps |
| Interview Coach | Generates interview questions |
| Career Strategist | Builds a 6-month roadmap |
