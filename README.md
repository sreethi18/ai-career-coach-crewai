# AI Career Coach using CrewAI

## Overview
This project is a multi-agent AI system built using CrewAI and OpenAI APIs.

The system helps users by:
- Analyzing resumes
- Suggesting suitable job roles
- Preparing interview questions
- Generating a 6-month career roadmap

## Technologies Used
- Python
- CrewAI
- OpenAI API
- Serper API
- GitHub

## Features
- Resume Skill Extraction
- Resume Improvement Suggestions
- Job Matching
- Interview Preparation
- Career Roadmap Generation
